## Project Overview
This is a **desktop calendar and to-do app** designed to help users manage tasks and events that **do not have to occur at a specific time**, offering a flexible scheduling solution. The app combines a **calendar interface** with a **task management system**, enabling users to plan their day, week, or month efficiently.

The project emphasizes **user-friendly design, clear task organization, and smooth interactions**. It is currently a **work-in-progress**, with ongoing improvements to the UI/UX and functionality.

**Prototype:** [Calendar App Link](https://calendar-desktop-app.vercel.app)

---

## Key Features
- **Calendar view** for yearly, monthly, weekly, or daily overviews  
- **Flexible task management**: add, edit, and remove tasks  
- **Event scheduling** without fixed times  
- **Interactive UI** designed for quick planning  
- **Responsive layout** for desktop users  

*Additional features are being developed, including notifications, task categories, and drag-and-drop task assignment.*

---

## Technologies & Tools
- **Frontend:** React, Tailwind CSS, Lucide-react icons  
- **Backend / Services:** Planned integration with Firebase for real-time storage  
- **Design:** Figma for UI/UX prototyping  
- **Other Tools:** Vite, Node.js, GitHub  

---

## Project Architecture
- `src/components/` → reusable UI components (Calendar, TaskCard, TaskList)  
- `src/pages/` → main views (Dashboard, Task Details, Calendar View)  
- `src/context/` → state management for tasks and calendar events  
- `src/api/` → planned API integration for persistent storage

## Setup Instructions
### Clone the repository
- `git clone https://github.com/IvaNikolova/calendar-desktop-app.git`
- `cd calendar-desktop-app/calendar-frontend`

### Install dependencies
Make sure you have Node.js 16+ installed, then run:
- `npm install`
  
### Start the development server
- `npm run dev`
Open your browser at `http://localhost:5173` to view the app.
