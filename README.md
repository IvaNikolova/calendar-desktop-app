## Setup Instructions
#### Clone the repository
- `git clone https://github.com/IvaNikolova/calendar-desktop-app.git`
- `cd calendar-desktop-app/calendar-frontend`
#### Install dependencies
Make sure you have Node.js (version 16 or higher recommended) installed. Then run:
- `npm install`
#### Start the development server
- `npm run dev`
